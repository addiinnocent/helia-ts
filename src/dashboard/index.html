API Routes (IPFS Standard)

GET /api/v0/id
Node identity: peer ID, multiaddresses, agent version
GET /api/v0/version
Helia version and environment info
GET /api/v0/swarm/peers
List connected peers with addresses and directions
GET /api/v0/swarm/addrs/local
Local multiaddresses this node is listening on
POST /api/v0/swarm/connect?arg=<multiaddr>
Connect to a peer by multiaddress
POST /api/v0/swarm/disconnect?arg=<multiaddr>
Disconnect from a peer
POST /api/v0/routing/provide?arg=<cid>
Announce content availability to DHT for peer discovery
GET /api/v0/routing/findprovs?arg=<cid>&num-providers=20
Find peers advertising a CID on the DHT (optional num-providers parameter)
POST /api/v0/add
Add content: multipart/form-data, JSON {"content":"..."}, or raw binary
GET /api/v0/cat?arg=<cid>
Retrieve content by CID (binary)
GET /api/v0/block/get?arg=<cid>
Get raw IPFS block by CID (binary)
POST /api/v0/block/put
Store a raw IPFS block (raw binary body)
GET /api/v0/block/stat?arg=<cid>
Get block metadata: CID and size
POST /api/v0/pin/add?arg=<cid>
Pin a CID
POST /api/v0/pin/rm?arg=<cid>
Unpin a CID
GET /api/v0/pin/ls
List all pinned CIDs
GET /api/v0/pin/ls?arg=<cid>
Check if CID is pinned
GET /api/v0/stats/bw
Bandwidth statistics (stub: Helia v6 does not expose bandwidth metrics)
Base: http://localhost:8081/api/v0
